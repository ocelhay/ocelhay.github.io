An App for a COVID-19 Model

Olivier
 2020-04-14  171 words   One minute 

The Covid-19 International Modelling Consortium (CoMo Consortium) is formulating a mathematical modelling response to help guide policymaking responses to the Covid-19 pandemic.

As a member of the CoMo Consortium, I have developed an interactive web application to provide easy access to a mathematical model of COVID-19 transmission. The interface contains all the features and functionalities needed to perform a model fit to the available data and investigate the potential impact of several non-pharmaceutical interventions. This can be accomplished in a few steps:

  1. providing data on the area/country of interest;
  2. customizing values of transmission related parameters;
  3. performing a visual calibration by adjusting the “Probability of infection given contact” and reporting rates;
  4. adding interventions that can be customized to occur at different times, with varying intensities;
  5. comparing between the model projections with different intervention measures  

 The App is deployed online on a server made available by Oxford Research Support Services and can be accessed at https://comomodel.net. The code can be accessed at https://github.com/ocelhay/como.
